# Light Summit — Official Platform

Official website for **Light Summit**, the first ever independent dialogue conference in Madinah, Saudi Arabia. The platform serves as the conference's information hub and delegate registration portal.

## What It Does

- **Registration Portal** — allows attendees to sign up and submit required information
- **Information Hub** — provides full details on the conference, its mission, sessions, and team
- **Schedule** — publishes the conference program and session timings
- **Delegate Resources** — guidance, requirements, and conference logistics

## Tech Stack

- **HTML** — structure and content
- **CSS** — styling and responsive layout
- **JavaScript** — interactivity and form handling

## Background

Light Summit was founded to create the first open, independent dialogue conference in Madinah — a platform for meaningful discussion and intellectual exchange that had never existed in the city before. This website was designed and built by the conference's founder to manage all digital aspects of the event.
